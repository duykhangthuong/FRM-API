Get trainee Notify History

By ThoTA

Endpoint

/api/Notify/history

Allow

trainee

Method

GET

Status

200 404

Parameter

email: email of trainee

PageSize: Position of page

PageNumber: Number of items perpage

SearchName: user(sender) and sendTo(reciever) field

Request Body
Response Body

{
    "totalRecords": int,
    "payload": [
        {
            "user": "string"
            "sendTo": "string"
            "message": "string"
            "createdAt": DateTime
            "isSeen": boolean
        }
    ]
}
